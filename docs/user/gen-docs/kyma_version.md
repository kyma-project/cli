# kyma version

Displays the version of Kyma CLI.

## Synopsis

Use this command to print the version of Kyma CLI.

```bash
kyma version
```

## Flags

```text
  -h, --help                    Help for the command
      --show-extensions-error   Prints a possible error when fetching extensions fails
```
